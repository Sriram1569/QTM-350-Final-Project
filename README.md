# QTM 350 Final Project

## Research Question
How have GDP per capita and employment rates co-moved in the USA and China (1990-2023)?

## Folder Structure
- data/raw: Original datasets
- data/processed: Cleaned datasets
- scripts: Python/SQL scripts
- notebooks: Analysis notebooks
- figures: Generated plots
- documentation: Codebook & project notes
- report: Quarto report (PDF & HTML)

## Instructions
1. Run `python scripts/fetch_clean_wdi.py` to fetch and clean data.
2. Run notebooks in `notebooks/` to generate analysis and figures.
3. Open `report/final_project.qmd` to render the PDF/HTML report.
